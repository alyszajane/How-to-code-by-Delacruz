Adding Images

To add an image to your webpage, use the <img> tag. The src attribute specifies the path to the image, and the alt attribute provides alternative text for the image.

Example of Adding an Image

    
<img src="image.jpg" alt="Description of image" width="300" height="200">

Adding Audio

To add audio to your webpage, use the <audio> tag. You can specify controls to allow users to play, pause, and adjust the volume.

Example of Adding Audio

     
<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  Your browser does not support the audio element.
</audio>

HTML Lists

HTML supports ordered and unordered lists. Use the <ul> tag for unordered lists and the <ol> tag for ordered lists.

Example of Unordered List

     
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

Example of Ordered List

    
<ol>
  <li>First Item</li>
  <li>Second Item</li>
  <li>Third Item</li>
</ol>

HTML Iframes

Iframes allow you to embed another HTML page within your current page. Use the <iframe> tag for this purpose.

Example of an Iframe

     
<iframe src="https://www.example.com" width="600" height="400"></iframe>

Semantic HTML

Semantic HTML uses HTML markup to reinforce the meaning of the content. Elements like <header>, <footer>, <article>, and <section> are examples of semantic elements.

Example of Semantic HTML

      
<header>
  <h1>My Website</h1>
</header>
<article>
  <h2>Article Title</h2>
  <p>This is an article about something interesting.</p>
